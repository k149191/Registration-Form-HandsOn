## Deskripsi Aplikasi
Aplikasi Registrasi Mahasiswa dan Mahasiswi merupakan aplikasi berbasis web yang digunakan untuk melakukan pendaftaran mahasiswa secara digital. Aplikasi ini dirancang untuk mempermudah proses pengumpulan, pengelolaan, dan penyimpanan data mahasiswa dengan sistem yang lebih terstruktur dan efisien. Secara keseluruhan, aplikasi ini membantu proses pendaftaran mahasiswa menjadi lebih praktis, terorganisir, dan mudah dikelola, sehingga dapat mengurangi penggunaan formulir manual dan mempermudah pengelolaan data peserta.
